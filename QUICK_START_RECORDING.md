## 📹 OBS QUICK SETUP

### **1. Install OBS**
```
Download: https://obsproject.com/
Install: Run installer, accept defaults
```

### **2. First-Time Setup**
```
Run OBS → Auto-Configuration Wizard
Select: "Optimize for recording"
Resolution: 1920x1080
FPS: 60
```

### **3. Add Sources**
```
Sources box → + button
Add "Display Capture" → OK
Add "Audio Output Capture" → Select speakers
```

### **4. Start Recording**
```
Controls box → "Start Recording"
Default location: Videos folder
Stop: Ctrl+F10
```

---

## ✂️ DAVINCI RESOLVE QUICK EDIT

### **1. Import**
```
Open DaVinci Resolve
File → Import Media → Select recording
Drag to timeline
```

### **2. Cut Mistakes**
```
Play video (Space)
Find mistake
Press B (blade) at start and end
Select bad section → Delete
```

### **3. Add Music**
```
Import music file
Drag to audio track 2
Lower volume to -25dB (right-click → Volume)
```

### **4. Add Title**
```
Effects → Titles → Text
Drag to timeline
Double-click to edit text
Type: "Welcome to KidCode Studio!"
```

### **5. Export**
```
Deliver page (rocket icon)
Format: MP4
Resolution: 1920x1080
Frame Rate: 60
Quality: 15000 Kb/s
Add to Render Queue → Start Render
```

## 🎯 PRO TIPS FOR FIRST VIDEO

### **Recording:**
1. **Speak 10% louder** than normal conversation
2. **Smile while talking** (sounds friendlier)
3. **Pause 2 seconds** after each sentence (easier editing)
4. **Do 3 takes** of intro (pick best one)

### **Editing:**
1. **Cut all "ums" and "uhs"**
2. **Remove long pauses** (>2 seconds)
3. **Keep it moving** (viewers have short attention)
4. **Add music** (makes it feel professional)

### **Uploading:**
1. **Use keywords in title** ("KidCode", "Tutorial", "AI")
2. **Add timestamps** in description
3. **Create playlist** ("KidCode Tutorials")
4. **Respond to ALL comments** (builds community)

## 🆘 TROUBLESHOOTING

### **Problem: Audio too quiet**
```
Fix: In OBS, increase mic volume
Or: Normalize in Audacity (Effect → Normalize)
```

### **Problem: Video laggy**
```
Fix: Lower resolution to 720p
Or: Reduce bitrate to 3000 Kbps
```

### **Problem: Mouse hard to see**
```
Fix: OBS → Right-click preview → Filters
Add: Cursor highlight (yellow circle)
```

### **Problem: Background noise**
```
Fix: Audacity → Effect → Noise Reduction
Select noise → Get Noise Profile → Apply
```
