## 💻 SOFTWARE SETUP

### **Screen Recording (FREE):**

#### **OBS Studio** (Recommended)
**Download:** https://obsproject.com/

**Setup:**
1. Install OBS Studio
2. Run Auto-Configuration Wizard
3. Set Canvas: 1920x1080, 60 FPS
4. Add Source → Display Capture
5. Add Source → Audio Output Capture
6. Settings → Output → Recording Path

**Recommended Settings:**
```
Base Resolution: 1920x1080
Output Resolution: 1920x1080
FPS: 60
Bitrate: 6000 Kbps
Encoder: NVIDIA NVENC (if available)
Format: MKV (convert to MP4 later)
```

#### **Alternatives:**
- **Camtasia** ($250) - Recording + editing
- **ScreenFlow** ($129, Mac only)
- **Loom** (FREE) - Quick recordings

### **Video Editing (FREE):**

#### **DaVinci Resolve** (Recommended)

**Pros:**
- Professional-grade

#### **Alternatives:**

## 🏠 RECORDING ENVIRONMENT

### **Room Setup:**
1. **Choose quiet room** (no fans, AC, traffic noise)
2. **Add soft surfaces** (pillows, blankets reduce echo)
3. **Close windows** (block outside noise)
4. **Turn off notifications** (phone, computer)

### **Lighting (for webcam):**
1. **Face the light** (window or lamp in front)
2. **Avoid backlight** (no windows behind you)
3. **Use soft light** (diffuse harsh lights)

### **Microphone Position:**
1. **6-8 inches from mouth**
2. **Slightly off-axis** (reduces plosives)
3. **Use pop filter** (or sock over mic!)

## 🎬 RECORDING PROCESS

### **Pre-Recording Checklist:**

- [ ] Script printed/open
- [ ] Water bottle nearby
- [ ] Phone on silent
- [ ] Notifications disabled
- [ ] Desktop clean (hide icons)
- [ ] KidCode Studio ready
- [ ] Test audio levels
- [ ] Test screen capture
- [ ] Do sound check

### **Recording Steps:**

#### **1. Intro (Record First)**
```
"Hey creators! Welcome to KidCode Studio!
I'm [Your Name], and today we're [tutorial topic]!
Let's get started!"
```

**Tips:**
- Smile and be energetic!
- Look at camera, not screen
- Speak 10% louder than normal
- Record 3 takes, pick best

#### **2. Screen Recording**
```
OBS Setup:
1. Start Recording (Ctrl+F9)
2. Navigate to KidCode Studio
3. Follow script step-by-step
4. Pause between sections (easier editing)
5. Stop Recording (Ctrl+F10)
```

**Tips:**
- Move mouse slowly and deliberately
- Click, then pause 1 second (helps editing)
- Zoom in for small UI elements
- Keep cursor in frame

#### **3. Voiceover (Optional)**
```
Record voiceover AFTER screen capture:
1. Play screen recording
2. Record voice in Audacity
3. Match narration to actions
4. Edit out mistakes
```

**Tips:**
- Read script while watching video
- Leave 2-second gaps between sentences
- Record in 30-second chunks (easier to redo)

#### **4. Outro (Record Last)**
```
"And that's how you [tutorial topic]!
If you found this helpful, subscribe for more!
Now go create something awesome! See you next time!"
```

## ✂️ EDITING WORKFLOW

### **DaVinci Resolve Setup:**

#### **1. Import Media**
```
File → Import Media
- Screen recording
- Webcam footage
- Music/sound effects
- Logo/graphics
```

#### **2. Create Timeline**
```
File → New Timeline
- Resolution: 1920x1080
- Frame Rate: 60 FPS
```

#### **3. Rough Cut**
```
1. Drag screen recording to timeline
2. Cut out mistakes (B key)
3. Remove long pauses
4. Trim start/end
```

**Keyboard Shortcuts:**
- `B` - Blade (cut)
- `A` - Select
- `Ctrl+Z` - Undo
- `Space` - Play/Pause

#### **4. Add B-Roll**
```
Insert:
- Close-ups of important clicks
- Reaction shots (if using webcam)
- Transition screens
- Text overlays
```

#### **5. Add Music**
```
1. Import background music
2. Lower to -25dB (under voice)
3. Fade in/out
4. Sync with intro/outro
```

**FREE Music Sources:**
- YouTube Audio Library
- Bensound.com
- Incompetech.com

#### **6. Add Text/Graphics**
```
Titles:
- Intro title (3 seconds)
- Step numbers
- Key points
- Outro with subscribe button

Effects:
- Zoom on important clicks
- Arrow pointers
- Highlight boxes
```

#### **7. Color Correction**
```
Basic Adjustments:
- Brightness: +5%
- Contrast: +10%
- Saturation: +15%
- Sharpness: +10%
```

#### **8. Audio Mixing**
```
Levels:
- Voice: -6dB to -3dB
- Music: -25dB to -20dB
- SFX: -10dB to -5dB

Effects:
- Noise Reduction (if needed)
- EQ (boost clarity)
- Compression (even levels)
```

## 🎙️ SCRIPT READING TIPS

### **Voice Techniques:**

1. **Smile while talking** (sounds friendlier)
2. **Stand up** (better breath support)
3. **Use hand gestures** (more energy)
4. **Pause between sentences** (easier editing)
5. **Emphasize key words** (more engaging)

### **Common Mistakes:**

❌ Speaking too fast
❌ Monotone voice
❌ Reading like robot
❌ No energy
❌ Filler words (um, uh, like)

✅ Speak slower than normal
✅ Vary pitch and tone
✅ Sound conversational
✅ Smile and be enthusiastic
✅ Edit out filler words

## 🆘 TROUBLESHOOTING

### **Audio Issues:**

**Problem:** Background noise
**Fix:** Noise reduction in Audacity

**Problem:** Echo/reverb
**Fix:** Record in softer room, add pillows

**Problem:** Too quiet
**Fix:** Move closer to mic, normalize audio

### **Video Issues:**

**Problem:** Choppy playback
**Fix:** Lower resolution, reduce bitrate

**Problem:** Mouse lag
**Fix:** Record at 60 FPS

**Problem:** Colors washed out
**Fix:** Color correction in editing
